# Instalare si Rulare

## Instalare

- Instalare Node.js (v20 LTS)
- `npm install`
- Comentare in `next.config.js` variabila `output` daca nu se va rula pe un device ce ruleaza Linux

## Rulare

```bash
# Production Build
npm start
```

```bash
# Development Build
npm run dev
```

```bash
# Build
npm run build
```
